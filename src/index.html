Echelon: Explained

A guide on the current and upcoming features of the app

What is Echelon?

Echelon is primarily a jukebox app that lets users add songs to a queue which will be played. Currently the users can pick any song in the Spotify library.

When feature complete, Echelon will have three distinct modes that a group can be set to. These modes include:

  • Jukebox Mode
  • DJ Mode
  • AllCast Mode

Each mode will be explained in detail on their respective pages.

How do I try out Echelon?

Echelon is currently in closed Alpha. We are planning on starting an open beta soon, but there is no definite date. If you are interested in joining the closed beta, you can click the button below to sign up.
